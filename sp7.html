Detail Normals

Developed with Haqvin Bager.
Provided with guidance from The Game Assembly. We went through their modelviewer
code to figure out how to proceed. The normals are supposed to blend into the underlying
normal map based on proximity from the camera.
float strengthMultiplier = 0.0f;
float dist = distance(eyePos, vertWorldPos);
strengthMultiplier = clamp((1.0f / (dist * dist)), 0.0f, DETAILNORMAL_DISTANCE_MAX); Additionally, a material should be able to use 4 different detail normals.
Which normal to use where is determined using the opacity of the material maps alpha channel

This was a requirment for the project; every group has some variation of it.
I mostly included it here cause I thought it was really cool.
